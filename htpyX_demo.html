htpyX Framework

A Modern Python Web Framework with W-Tech Integration

What is htpyX?

htpyX is a powerful Python web framework that combines the simplicity of Python with modern web development practices. It features W-Tech integration, a unique approach to handling web components and state management.

CSS Generator

Build dynamic CSS with Python syntax:

builder = CSSBuilder()
builder.var("primary", "#4a90e2")
       .define(".button", 
           background="var(--primary)",
           padding="10px 20px",
           border_radius="4px"
       )

HTML Elements

Create HTML elements with Python functions:

Div(
    H1("Welcome to htpyX"),
    P("Modern web development with Python"),
    className="container"
)

W-Tech Integration

Manage state and reactivity with W-Tech:

counter = w(0, "counter")
Button("Increment", 
       onclick=f"{counter.name}++")

Key Features

  • Pythonic Syntax: Write web applications using familiar Python syntax
  • W-Tech Integration: Built-in state management and reactive updates
  • Modern CSS Tools: Advanced CSS generation with variables and nested rules
  • Component-Based: Create reusable web components easily
  • Type Safety: Leverage Python's type system for better development experience
